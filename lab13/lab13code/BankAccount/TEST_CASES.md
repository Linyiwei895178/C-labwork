# BankAccount 测试用例

## 测试目标
验证BankAccount系统的各项功能是否正常工作

## 测试环境
- 操作系统: Windows 10/11
- 编译器: Visual Studio 2022
- 构建工具: CMake 3.10+

## 测试用例

### 测试用例1: 基本账户操作
**测试步骤:**
1. 运行程序
2. 输入命令: `a s 123456 0.05` (添加储蓄账户)
3. 输入命令: `d 0 1000 salary` (存款1000元)
4. 输入命令: `w 0 200 shopping` (取款200元)
5. 输入命令: `s` (显示账户信息)

**预期结果:**
- 成功创建储蓄账户
- 成功存款和取款
- 显示正确的账户余额

### 测试用例2: 交易查询
**测试步骤:**
1. 运行程序
2. 输入命令: `a s 123456 0.05` (添加储蓄账户)
3. 输入命令: `d 0 1000 salary` (存款1000元)
4. 输入命令: `q 2008-11-01 2008-11-30` (查询交易记录)

**预期结果:**
- 显示所有交易记录
- 包括存款和账户创建记录

### 测试用例3: 多账户管理
**测试步骤:**
1. 运行程序
2. 输入命令: `a s 123456 0.05` (添加储蓄账户)
3. 输入命令: `a c 789012 5000 0.001 100` (添加信用账户)
4. 输入命令: `d 0 1000 salary` (给储蓄账户存款)
5. 输入命令: `s` (显示所有账户)

**预期结果:**
- 成功创建两种类型的账户
- 显示所有账户的详细信息

## 运行测试的方法

### 方法1: 使用预设命令文件
1. 确保`commands.txt`文件存在
2. 运行程序: `.\BankAccount\build\Debug\bank.exe`
3. 程序会自动执行commands.txt中的命令

### 方法2: 交互式测试
1. 运行程序: `.\BankAccount\build\Debug\bank.exe`
2. 根据提示输入命令:
   - `(a)add (d)deposit (w)withdraw (s)show (c)day (n)next month (q)query (e)exit`

## 常见命令格式

- 添加储蓄账户: `a s 账户ID 年利率`
- 添加信用账户: `a c 账户ID 信用额度 日利率 年费`
- 存款: `d 账户索引 金额 描述`
- 取款: `w 账户索引 金额 描述`
- 显示账户: `s`
- 查询记录: `q 开始日期 结束日期`
- 退出: `e`
